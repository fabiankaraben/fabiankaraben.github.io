### Proyecto 1: VeoBible

- **Título:** VeoBible — Aplicación Web Progresiva Offline-First
- **Categoría:** Desarrollo Web
- **URL:** <https://veobible.com>
- **GitHub:** <https://github.com/xeost/veobible-app>
- **Descripción:** Una Aplicación Web Progresiva (PWA) moderna para leer la Biblia. Generada estáticamente, completamente funcional sin conexión y desplegada en el edge. Incluye UI bilingüe, marcadores de texto, sesiones de lectura optimistas y ~2,400 páginas pre-renderizadas por SSG. El backend es una API REST para sincronización construida en Cloudflare Workers con Hono y D1 (SQLite en el edge). Incorpora Service Worker con Cache API para soporte completo sin conexión y autenticación segura con verificación de JWTs vía Supabase Auth.
- **Tecnologías:** Next.js 14, TypeScript, Tailwind CSS, Cloudflare Workers, Hono, D1 (SQLite), Supabase Auth, PWA

### Proyecto 2: Wordsus

- **Título:** Wordsus — Plataforma de lectura online gratuita
- **Categoría:** Desarrollo Web
- **URL:** <https://wordsus.com>
- **GitHub:** <https://github.com/wordsus/wordsus-static-site>
- **Descripción:** Plataforma de lectura online gratuita para libros y cursos educativos. Construida con una aplicación Next.js 16 con App Router exportada estáticamente hacia Cloudflare Pages para máximo rendimiento. Incluye una API REST ultrarrápida usando Hono desplegada en V8 isolates en el edge a través de Cloudflare Workers. Ofrece sincronización multi-dispositivo del progreso y favoritos usando base de datos D1 (SQLite en el edge), soporte i18n, búsqueda full-text, temas claros/oscuros, seguimiento de lectura y soporte para audio/video.
- **Tecnologías:** Next.js 16, TypeScript, Tailwind CSS, Cloudflare Pages, Cloudflare Workers, Hono, D1 (SQLite), Supabase Auth

### Proyecto 3: Staticl10n CLI

- **Título:** Staticl10n CLI — Herramienta de traducción automática de sitios web
- **Categoría:** Desarrollo de Software / Herramienta CLI
- **GitHub:** <https://github.com/xeost/staticl10n-cli>
- **Descripción:** Herramienta CLI para la traducción automática de sitios web estáticos. Captura, procesa y traduce sitios web completos de extremo a extremo utilizando LLMs locales (Ollama). Cuenta con una interfaz CLI interactiva construida con Inquirer y Chalk, un crawler headless con Playwright para descubrimiento profundo de URLs y exportación estática, traducciones semánticas de HTML que mantienen intacta la estructura del DOM, y rastreo robusto de estado con better-sqlite3 y sistema de caché de memoria de traducción.
- **Tecnologías:** Node.js, TypeScript, SQLite, Playwright, Cheerio, Ollama
